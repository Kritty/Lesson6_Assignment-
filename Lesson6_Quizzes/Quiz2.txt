1.  Name two methods you can use to add a node to the DOM tree, and explain the difference between them.

Two methods are:
appendChild()
insertBefore()

The insertBefore() lets a user specify and organize the order of inserted nodes among sibling nodes. 
appendChild is put after existing child nodes.
